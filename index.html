portfolio

JORTS: Rails App

Ruby on Rails & Google Maps API

Focus: object oriented programming

JORTS is a location based app that allows users to find others in their area to do activities with. This was a final group assignment. I worked with another developer to create a Rails app that implemented all the things we've learned in the course. From collaborative software development to databases and CSS, we created JORTS in 6 days. JORTS specifically employed the google maps API to give users the capabilties to search activities in their area and create activities with specific locations.

The Jokebox: Javascript Jukebox

Javascript & Scss

Focus: object oriented programming

The assignment was to utilize Javascript, jQuery, HTML and Scss to develop a jukebox that could play, load, pause, stop, go to the next track and pick a random track. We themed this around a comedy club. The 'Jokebox' was created so people can hear great stand-up bits from some of their favorite comedians!

Seadogs: Rails App

Ruby on Rails

Focus: databases and validations

We were asked to create a project centered around the needs of shippers and to specifically showcase our proficiency with validations and join tables in our database. We catered ours to pirates and created a bit of a guessing game in the process. We pushed ourselves beyond the requirements by implementing a dynamic map for users to interact with.

Birderz: Rails Blog

Ruby on Rails

Focus: databases and MVC

This was our first step into Rails! We were asked to create a blog where users could log in, log out, create posts, comment on other's posts, etc. We wanted our blog to be a bit silly and have a specific niche so we chose to create a space for bird-watchers to connect with eachother! Users can sign-up, log in, create a profile, edit their profile, write a post, view other users' posts (on a feed), comment on posts, sign out and delete their accounts.
